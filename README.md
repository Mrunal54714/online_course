🚀 LearnAI: 8-Week Masterclass Landing Page
A clean, high-conversion landing page for an online AI course. This project demonstrates modern web design principles, including a responsive navigation bar, hero sections, and data visualization using Bootstrap progress bars.

🖼️ Features
Responsive Hero Section: High-impact "above-the-fold" content with a clear Call-to-Action (CTA).

Curriculum Progress Tracker: Uses dynamic Bootstrap Progress Bars to visually represent the course journey.

Instructor Profile: A professional bio section with rounded image styling and aligned typography.

Enrollment Form: A fully styled, responsive grid-based form with validation-ready inputs.

Custom Styling: Overrides Bootstrap defaults with custom CSS for a unique brand identity (dark/light contrast).

🛠️ Tech Stack
HTML5: Semantic structure for SEO and accessibility.

CSS3: Custom styles for layout, hover effects, and background image management.

Bootstrap 5.3: Utilized for the responsive grid system, navigation, and progress components.

jQuery: Included for DOM manipulation and future interactive features.
